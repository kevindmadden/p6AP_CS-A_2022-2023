Project #02 - Animate Something

Description:

Create a unique animation that responds in some way both to mouse clicks and mouse hovering.

Requirements:

  1. Your animation must react in two different ways. In other words, your animation should not respond in the exact same way to both mouse clicking versus hovering.
  2. A reaction can be pretty much anything that is visible on the screen. It can be as simple as changing a color.
  3. Experiment around with what you can do with the toolset you have been given so far. Take your time to figure out how everything works together.
  4. Your animation should be coded in Java using StdDraw.

Reminder:

  • You will need to use if-statements in the animation loop so that certain parts of your code gets run only when certain conditions are met.

Questions?

Q: Can I have a group of three?
  • No. Two is the max.
    • Q: How do I submit as a group? Q: How do I do something involving mouse clicking? You only showed us how to do stuff involving mouse hovering?
  • See StdDraw documentation.
    • Q: How do I generate a random number?
  • Call Math.random(), which returns a random double value between 0 and 1.
